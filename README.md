# pokémon_hash_map

To learn more about data management, analysis and structuring I decided to make my own (basic) hash mapping project.
The goal is simple: give the progam a unique identifier between 001 and 151 and the program will tell you the corresponding pokémon in a hash mapping manner, showing the name and typing of the pokémon.

Also it is possible to add more pokémon if looking for an ID not yet existing in the hash map.
